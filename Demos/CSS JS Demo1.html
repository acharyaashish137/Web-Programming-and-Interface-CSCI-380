Using Bluefish Text Editor

Bluefish is a powerful editor targeted towards programmers and web developers,
with many options to write websites, scripts and programming code.
It is easy to write HTML, CSS and JavaScript as it has various auto-generated code.

Advantages of using Bluefish

  1. Bluefish include spell checking and automated content recovery
  2. Bluefish is easy to use.

Drawbacks of Bluefish as HTML Editor

  • It has no syntax checker or beautifier.
  • It is young and still missing specialised features.
  • It is not ideal for anyone who'd like a visual editor.
